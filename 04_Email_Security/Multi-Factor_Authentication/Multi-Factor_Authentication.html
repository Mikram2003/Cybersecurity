Multi-Factor_Authentication

  1. Definition: An authentication method that requires two or more verification factors to access a resource such as an application, online account, or VPN.

  2. Something You Know: The first factor, which is typically a password or PIN.

  3. Something You Have: The second factor, which can be a smartphone, hardware token, or smart card.

  4. Something You Are: The third factor, which includes biometrics like fingerprints, facial recognition, or retinal scans.

  5. Enhanced Security: Provides a higher level of security compared to single-factor authentication by combining multiple factors.

  6. Authentication Apps: Uses mobile apps like Google Authenticator or Microsoft Authenticator to generate verification codes.

  7. SMS Verification: Sends a verification code via text message to the user's registered mobile number.

  8. Push Notifications: Sends a prompt to the user's mobile device to approve the login attempt.

  9. Biometric Verification: Uses biometric data for authentication, such as fingerprint or face ID.

  10. Risk-Based Authentication: Adjusts the required authentication factors based on the perceived risk of the login attempt.
